## Шаг 1: Инициализация проекта

### Созданные/изменённые файлы
- `go.mod` — модуль `rag-pipeline`, Go 1.25, три основные зависимости
- `go.sum` — контрольные суммы зависимостей (генерируется автоматически)

### Ключевые решения
- Имя модуля: `rag-pipeline`
- `github.com/qdrant/go-client v1.13.0` — официальный gRPC-клиент Qdrant
- `github.com/anthropics/anthropic-sdk-go v1.3.0` — официальный SDK для Claude API
- `golang.org/x/sync v0.20.0` — `errgroup` и семафор для управления параллелизмом в эмбеддере
- Транзитивные зависимости: `grpc v1.66.0`, `protobuf v1.34.2`, `tidwall/gjson` (JSON в SDK Anthropic)

### Известные ограничения
- `go.sum` добавлен в `.gitignore` — при клонировании репозитория потребуется `go mod tidy` для его воспроизведения
- Версия Go 1.25 требует актуального тулчейна; на более старых версиях может не собраться
